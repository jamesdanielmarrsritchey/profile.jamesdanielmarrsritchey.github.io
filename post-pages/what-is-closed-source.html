Definition of Closed Source


Closed source (AKA proprietary) is a state of legal inaccess, created by Copyright law, and restrictive licensing terms, applicable to creative works. If a work is closed source this means the source code, or source materials are usually not available to the public, and any product intended for endusers is licensed using restrictive terms.
